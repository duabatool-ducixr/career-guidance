Project Title: Career Guidance System Using C++
<br>
Purpose: To help students identify suitable career paths based on their interests, skills, and preferences.
<br>
Problem Statement: Many students are unsure about which career field matches their interests and abilities. This system provides guidance by asking a series of questions and recommending careers accordingly.
<br>
Objective: To develop a menu-driven C++ application that analyzes user responses and suggests career options.
